# Widget Grid
## a four-in-one widget app built with vanilla javascript 
---
**the grid:**
1. digital clock
* pomodoro timer
* set alarm
* classic timer up 
* contdown timer 
2. weather app 
* get your local weather
* search any US city
* see upcoming hourly weather
3. calculator 
4. notepad
---
the perfect tool to have open on your secondary monitor when doing coding challenges!
